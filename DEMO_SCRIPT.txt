[0:00 — HOOK]
Traveling should be exciting, but planning it often feels like a second job. Between matching flights, checking safety advisories, converting currencies, and sticking to a budget, you end up juggling dozens of tabs. That's why we built Trip Planner.

[0:20 — WHAT IT IS]
Trip Planner is an intelligent travel concierge built on the Google Agent Development Kit and the Model Context Protocol. It acts as an automated, secure planner that takes your trip destination, duration, and budget, coordinates specialized sub-agents, pulls real-time local statistics, and generates a personalized travel itinerary.

[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's look at the workflow. When a user sends a prompt, it doesn't go straight to the model. First, it hits our Security Checkpoint. If it passes, it moves to the central Orchestrator. The Orchestrator splits the task: it calls the Destination Agent to gather top spots and travel tips, and the Budget Advisor to cross-check costs. Both agents use our Model Context Protocol, or MCP server, to retrieve live mock reviews, exchange rates, and flight costs. Finally, if the budget is too low, we trigger a Human-in-the-Loop pause asking for user confirmation before compiling the final itinerary.

[3:00 — SECURITY & MCP]
This secure design is made possible by our security checkpoint node, which scrubs emails and phone numbers and blocks bad intent. Our MCP server acts as the agent's eyes and ears, exposing tools to estimate flights, check safety levels, and fetch exchange rates locally.
